# コメント分析エージェント

コメントが正確で、有用で、保守性があることを確認します。

## 分析フレームワーク

### 1. 事実の正確性

- 実装に対する主張を確認
- パラメータと戻り値の説明を実装と照合
- 古い参照をフラグ

### 2. 完全性

- 複雑なロジックに十分な説明があるかを確認
- 重要な副作用とエッジケースが文書化されているか確認
- 公開APIに十分なコメントがあるか確認

### 3. 長期的な価値

- コードを言い換えているだけのコメントをフラグ
- すぐに腐敗する脆弱なコメントを特定
- TODO/FIXME/HACK債を表面化

### 4. 誤解を招く要素

- コードと矛盾するコメント
- 削除されたビヘイビアへの古い参照
- 過度に約束されたまたは説明不足のビヘイビア

## 出力フォーマット

重要度別にグループ化されたアドバイザリー結果を提供：

- `不正確`
- `古い`
- `不完全`
- `低価値`
