# 07_comparisons — 比較分析

ECC の設計判断を、他のハーネスや公式ドキュメントと比較して相対化する場所。
「ECC流が当たり前」にならないための視点を持つ。

## 比較対象の例

- Claude Code 公式ドキュメント・サンプル
- 他の OSS エージェントハーネス
- 自分の既存の開発スタイルとの差分

## ファイル命名規則

```
{テーマ}_comparison.md

例:
  agent-routing_comparison.md    → ルーティング設計の比較
  skill-format_comparison.md     → スキルフォーマットの比較
```

## ファイル構成テンプレート

```markdown
# {テーマ} 比較分析

**比較日**: YYYY-MM-DD

## ECC のアプローチ
（ECCがどう解決しているか）

## 他のアプローチ
（比較対象がどう解決しているか）

## 差分・気づき
（何が違うか、なぜ違うか）

## 自分の結論
（自分がハーネスを作るならどちらを選ぶか、なぜか）
```
