# ポモドーロタイマー + タスク管理 設計仕様書
## Haiku-Opus パターン実装版

---

## 概要

**ポモドーロ・テクニック（25分集中 + 5分休憩）とシンプルなタスク管理を統合したウェブアプリケーション**。

このアプリケーションは、時間管理とタスク優先度を同時に処理し、開発者やナレッジワーカーが効率的に仕事を進められるよう支援します。HTML/CSS/JavaScript のみで実装され、LocalStorage を活用してタスクとセッション記録をクライアント側で永続化します。ユーザーは開始/一時停止/リセットの簡潔なインタフェースで、ポモドーロサイクルを管理でき、その際タスクリストから現在のタスクを選択して進捗管理できます。

## 技術スタック（推奨）

### 必須スタック
- **HTML5** — セマンティックマークアップ
- **CSS3** — Flexbox/Grid、CSS変数（テーマカラー管理）、アニメーション
- **JavaScript (Vanilla)** — タイマーロジック（setInterval）、タスク状態管理、LocalStorage API
- **LocalStorage** — タスク、セッション記録、ユーザー設定の永続化

### 推奨パターン
- **状態管理**: クロージャまたは ES6 Class で App クラスを定義
- **イベント駆動**: `addEventListener` で UI イベントを管理
- **日時管理**: `Date` オブジェクトで日付判定（日付変更検出）
- **通知**: Web Audio API（音声）+ `alert()` / `console.log()` （または Notification API）

### ファイル構成
```

│   ├── timer-end.mp3 — タイマー終了音（オプション）
│   └── break-end.mp3 — 休憩終了音（オプション）
```

   - Timer Display（MM:SS 形式）
   - Status Indicator（集中中/休憩中/停止中）
   - Start / Pause / Reset ボタン

   - Task List（完了/削除ボタン付き）

## エッジケース & 対応策

| ケース | 説明 | 対応 |
|-------|------|------|
| **長時間離席** | タイマー実行中に24時間以上放置 | セッション日時を記録、日付変更時に新規セッション扱い |
| **ブラウザタブ非アクティブ** | `setInterval` の精度低下（ブラウザ最適化） | `requestAnimationFrame` への移行検討、再度アクティブになった時の修正カウント |
| **LocalStorage 満杯** | クォータ超過エラー | try-catch でキャッチ、古いタスクを自動削除またはユーザーに警告 |
| **タイマー中のブラウザ閉鎖** | セッション継続時間の記録ロス | 最後の更新時刻を保存、再起動時に経過時間を計算して復元 |
| **複数タブでの同時実行** | 複数タブで同一 LocalStorage にアクセス | `storage` イベントリスナーで同期、または複数実行禁止フラグ |
| **システム時刻の巻き戻し** | OS のシステム時刻が前に戻された | 常に `Date.now()` で経過時間を相対計算、絶対時刻に依存しない |
| **スマートフォン画面回転** | 縦横回転時のタイマー表示ズレ | viewport meta tag + media query で対応、Flexbox で自動調整 |
| **モバイルブラウザのバックグラウンド制限** | iOS Safari がタイマーを一時停止 | 正確な時刻管理のため `visibilitychange` イベントで「一時停止中」と判定 |
| **音声再生ブロック** | Autoplay Policy により音が鳴らない | ユーザーインタラクション後に音声キャッシュ、通知は Notification API で代替 |
| **ネットワーク不要の明示** | ユーザーがオフラインで使えるか不明 | README に「オフライン完全対応」と明記、Service Worker は不要 |

## 参考: LocalStorage スキーマ設計

```javascript
{
  "tasks": [
    { id, title, completed, priority, createdAt, completedAt },
    ...
  ],
  "sessionCount": number,       // 本日のセッション完了数
  "lastSessionDate": string,    // YYYY-MM-DD
  "totalSessions": number,      // 歴史的なセッション総数
  "theme": "light" | "dark",
  "settings": {
    "timerWork": 1500,          // 秒数（デフォルト25分）
    "timerBreak": 300,          // 秒数（デフォルト5分）
    "longBreakInterval": 4,     // 4セッション後に長休憩
    "notificationEnabled": boolean
  }
}
```
