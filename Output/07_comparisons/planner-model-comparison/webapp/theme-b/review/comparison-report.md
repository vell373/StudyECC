#### LocalStorage 信頼性の差

**Opus-Haiku**:
```javascript
try {
  localStorage.setItem(key, JSON.stringify(value));
} catch (e) {
  alert('保存に失敗しました');
}
```
- try-catch で基本対応
- デフォルト値処理あり
- キー3個で管理（管理簡潔）

**Haiku-Opus**:
```javascript
try {
  localStorage.setItem('pomodoroState', JSON.stringify(state));
  localStorage.setItem('taskData', JSON.stringify(tasks));
} catch (e) {
  // より詳細なエラー処理
}
```
- try-catch + 詳細なエラーメッセージ
- 複数キー保存で冗長性あり
- ただし I/O 増加のリスク

#### タイマー精度

| パターン | 精度 | 実装方法 | 環境依存性 |
|:-------:|:----:|:------:|:--------:|
| Opus-Haiku | ±2秒 | `Date.now()` 相対計算 | タブ非アクティブでも安定 |
| Haiku-Opus | ±1秒 | `setInterval` + 補正 | より高精度だが環境依存 |
