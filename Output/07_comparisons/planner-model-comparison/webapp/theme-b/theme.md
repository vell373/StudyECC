# テーマB: ウェブアプリ開発

## タイトル
「ポモドーロタイマー + タスク管理」

## 背景・目的
ポモドーロ・テクニック（25分集中、5分休憩を繰り返すタイムマネジメント法）は、多くの開発者に使われています。
このタスクでは、タイマー機能とシンプルなタスクリスト管理を組み合わせたウェブアプリを作成してください。
実装は HTML/CSS/JavaScript のみで、ローカルストレージを使ってタスクを保存します。

## 想定ユーザー
- ポモドーロ・テクニックで作業時間を管理したい人
- タスク管理をシンプルに行いたい人
- ローカルで動作するツールを求める人

## 要求事項

### Must-Have（必須）
1. **ポモドーロタイマー** — 25分集中、5分休憩の自動サイクル
   - 受け入れ条件: 
     - 開始/一時停止/リセットボタンが動作する
     - 時間がカウントダウン表示される
     - 25分終了時と5分終了時に通知（音またはアラート）が出る
2. **セッションカウンター** — 何回ポモドーロを完了したかを表示
   - 受け入れ条件: 25分セッション終了時に +1 カウント、4セッション後に長休憩（15分）を提案
3. **タスクリスト** — タスクを追加、完了、削除できる
   - 受け入れ条件:
     - 新規タスクを入力・追加できる
     - タスク完了時にチェックボックスで取り消し線表示
     - 削除ボタンで削除可能
4. **ローカルストレージ保存** — タスク と セッションカウントを保存
   - 受け入れ条件: ブラウザを閉じて再度開いた時、タスク一覧とカウントが残っている

### Should-Have（あると良い）
1. **タスク優先度設定** — 高/中/低 の優先度を設定できる
2. **今日のセッション記録** — 本日完了したセッション数と完了したタスク数を表示
3. **ダークモード** — ライト/ダークテーマの切り替え
4. **進捗表示** — ポモドーロセッションの進捗をビジュアル表示（プログレスバー等）

### Nice-to-Have（実装できれば）
1. **タスク編集** — タスク名を編集できる
2. **カテゴリ分け** — タスクをカテゴリで分類
3. **統計情報** — 週の完了セッション数グラフ等
4. **通知レベル設定** — 音量調整、通知方法の選択

## 技術制約
- **言語**: HTML, CSS, JavaScript のみ
- **外部ライブラリ**: 不可（タイマーライブラリ等）
- **ストレージ**: LocalStorage のみ（サーバー不要）
- **動作環境**: モダンなブラウザで動作すること

## 出力ファイル構成
```
output/
├── index.html      — メインファイル（HTMLとJavaScript）
├── style.css       — スタイルシート
├── script.js       — JavaScript（外部化する場合）
└── README.md       — 使い方（起動方法、操作説明）
```

## 評価の観点
- ポモドーロタイマーが正確に動作するか（25分、5分）
- セッションカウント、タスク管理の正確性
- ローカルストレージ機能が正しく動作するか
- UIが使いやすく、分かりやすいか
- コード品質（関数分割、状態管理の明確さ等）

## 完成イメージ
実際に集中力が必要な作業時に、このアプリを使って「タスク管理」と「時間管理」を同時に行える、実用的なツール。
毎日開いて使いたくなるシンプルで洗練されたUI。
