# Markdown Live Preview Editor

外部ライブラリに依存せず、ブラウザだけで動作するマークダウンエディターです。
左ペインに入力すると、右ペインにリアルタイムでプレビューが表示されます。
入力内容は LocalStorage に自動保存され、ページをリロードしても復元されます。

## 起動方法

ビルド不要・サーバー不要です。`output/index.html` をブラウザで直接開いてください。

```
# Mac
open output/index.html

# Windows
start output\index.html

# Linux
xdg-open output/index.html
```

ローカル HTTP サーバー（任意）:

```
# Python 3
python3 -m http.server 8080
# → http://localhost:8080/output/ にアクセス
```

## 機能一覧

### Must-Have
- マークダウン入力（複数行・改行保持）
- リアルタイムプレビュー（100ms デバウンス）
- 基本マークダウン解析: 見出し / 段落 / 太字 / イタリック / リスト / インラインコード / コードブロック / リンク
- LocalStorage への自動保存と復元

### Should-Have
- ライト / ダークモード切替（設定保存）
- 入力フォント選択（Courier Prime / Lato / Noto Sans JP）
- プレビューの整ったスタイリング（H1～H6 の階層、コードブロック、引用、テーブル等）

### Nice-to-Have
- テーブル記法対応（GFM スタイル）
- HTML コピー（Clipboard API + フォールバック）
- リセットボタン（confirm 確認付き）

## 対応マークダウン記法

| 記法 | 例 |
| --- | --- |
| 見出し | `# H1` ～ `###### H6` |
| 段落 | 空行で分離 |
| 太字 | `**text**` または `__text__` |
| イタリック | `*text*` または `_text_` |
| インラインコード | `` `code` `` |
| コードブロック | ` ```lang ` ～ ` ``` ` |
| 順序なしリスト | `- item` / `* item` / `+ item` |
| 順序付きリスト | `1. item` |
| リンク | `[text](https://example.com)` |
| 引用 | `> 引用文` |
| 水平線 | `---` / `***` / `___` |
| テーブル | `| col | col |` + 区切り行 |

注意: ネストしたリストや、太字の中のイタリックなど高度なネストは初版では未対応です。

```
output/

├── style.css    # スタイル（CSS カスタムプロパティでテーマ切替）
```

## 動作確認ブラウザ

Chrome / Firefox / Safari / Edge の最新版を想定しています。
LocalStorage を無効化したプライベートモードでも、自動保存だけがサイレント失敗し、
他の機能はそのまま動作します。

## 設計メモ

- **パーサーは正規表現ベース**: ブロック要素 → インライン要素の 2 段階で処理。
  コードブロックとインラインコードは事前にプレースホルダで退避し、
  内部のマークダウンが誤変換されないようにしています。
- **XSS 対策**: 入力全体を最初に HTML エスケープしてからパースし、
  `javascript:` / `data:` スキームのリンクは `#` に置換しています。
- **デバウンス**: 入力イベントごとに即時パースせず、100ms 待ってから再描画します。
- **LocalStorage**: `try/catch` で保護し、QuotaExceeded やプライベートモードでも
  例外を握りつぶしてアプリは動作継続します。

## ショートカット

- 入力エリア内でタブキーは標準の動作（フォーカス移動）です。
- リセットは confirm ダイアログで誤操作を防止します。
