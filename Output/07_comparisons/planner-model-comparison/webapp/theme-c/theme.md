# テーマC: ウェブアプリ開発

## タイトル
「シンプルな習慣トラッカー」

## 背景・目的
毎日繰り返したい習慣（運動、読書、瞑想など）を追跡し、連続実施日数をカウントするツールです。
このタスクでは、習慣ごとに「今日実施した」をマークできるシンプルなトラッカーを作成してください。
連続日数を視覚的に表示し、モチベーション維持をサポートします。

## 想定ユーザー
- 毎日の習慣を記録・追跡したい人
- 連続実施日数をモチベーション源にしたい人
- ローカルで動作するツールを求める人

## 要求事項

### Must-Have（必須）
1. **習慣の追加** — ユーザーが習慣を定義・追加できる
   - 受け入れ条件:
     - テキスト入力で習慣名を入力できる
     - 「追加」ボタンで習慣がリストに加わる
     - 最大10個程度の習慣を管理可能
2. **日々のチェックイン** — 習慣を実施した日をマークできる
   - 受け入れ条件:
     - 習慣ごとに「今日実施」ボタンがある
     - ボタン押下で本日の実施記録が保存される
     - 1日1回のみカウント（重複チェック防止）
3. **連続日数の表示** — 各習慣の連続実施日数を表示
   - 受け入れ条件:
     - 連続日数が大きく見やすく表示される
     - 記録がない日が挟まると連続日数がリセットされ 0 になる
     - 最高連続日数も記録・表示される
4. **ローカルストレージ保存** — 習慣と記録をブラウザに保存
   - 受け入れ条件: ページリロードやブラウザ再起動後も記録が残っている

### Should-Have（あると良い）
1. **習慣削除** — 不要な習慣を削除できる
2. **カラーコード** — 習慣ごとに色を選択できる（視覚的識別を助ける）
3. **週ビュー** — 過去7日間の実施状況をカレンダー表示
4. **統計サマリー** — 本週の実施率（100%なら「完璧な週！」等）を表示

### Nice-to-Have（実装できれば）
1. **削除確認ダイアログ** — 習慣削除時に確認を求める（誤削除防止）
2. **習慣名編集** — 習慣名を後から変更できる
3. **スナップショット機能** — 月単位の記録を表示
4. **ダークモード** — ライト/ダーク切り替え

## 技術制約
- **言語**: HTML, CSS, JavaScript のみ
- **外部ライブラリ**: 不可
- **ストレージ**: LocalStorage のみ
- **日付処理**: JavaScript 標準の Date オブジェクトのみ（moment.js など外部ライブラリは不可）
- **動作環境**: モダンなブラウザで動作すること

## 出力ファイル構成
```
output/
├── index.html      — メインファイル（HTMLとJavaScript）
├── style.css       — スタイルシート
├── script.js       — JavaScript（外部化する場合）
└── README.md       — 使い方（起動方法、操作説明、日付の扱い等）
```

## 評価の観点
- すべてのMust-Have要件が実装されているか
- 日付管理（連続日数カウント）が正確か
- UIが見やすく、直感的に操作できるか
- ローカルストレージが正しく機能するか
- コード品質（関数分割、状態管理の明確さ等）

## 完成イメージ
毎朝起きてアプリを開き、昨日の習慣をチェック。連続日数が増えていく快感を感じながら、
自分の成長を可視化できるツール。
シンプルだが、使い続けたくなるUI設計。
