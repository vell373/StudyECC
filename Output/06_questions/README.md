# 06_questions — 疑問リスト

ECC を読んでいて湧いた「なぜ？」「どうして？」を蓄積する場所。
未解決のまま残しておき、理解が深まったら解決済みに移動する。

## ファイル構成

```
06_questions/
├── README.md           （このファイル）
├── open/               （未解決の疑問）
└── resolved/           （解決済みの疑問）
```

```markdown

**関連**: ECC/{ファイルパス} または 04_patterns/{パターン名}.md

（具体的に何が分からないか）

## 現時点の仮説
（今の時点での自分の推測）

## 解答

**参照**: （答えが見つかったソース）
```
