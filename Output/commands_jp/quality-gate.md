# 品質ゲートコマンド

オンデマンドでファイルまたはプロジェクトスコープに対して ECC 品質パイプラインを実行します。

## 使用方法

`/quality-gate [path|.] [--fix] [--strict]`

- デフォルトターゲット: 現在のディレクトリ（`.`）
- `--fix`: 設定に従って自動フォーマット・修正を許可
- `--strict`: サポートされている箇所で警告でも失敗

## パイプライン

1. ターゲットの言語・ツールを検出
2. フォーマッターチェックを実行
3. 利用可能な場合、lint・型チェックを実行
4. 簡潔な改善リストを生成

## 注記

このコマンドはフック動作をミラーリングしますが、オペレータ（ユーザー）が手動で実行するバージョンです。

## 引数

$ARGUMENTS:
- `[path|.]` オプション：ターゲットパス
- `--fix` オプション
- `--strict` オプション
