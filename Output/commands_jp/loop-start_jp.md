# ループ開始コマンド

安全なデフォルトを持つ管理された自律ループパターンを開始します。

## 使用方法

`/loop-start [pattern] [--mode safe|fast]`

- `pattern`: `sequential`、`continuous-pr`、`rfc-dag`、`infinite`
- `--mode`:
  - `safe`（デフォルト）: 厳密な品質ゲートとチェックポイント
  - `fast`: スピード向け削減ゲート

## フロー

1. リポジトリ状態とブランチ戦略を確認します。
2. ループパターンとモデル層戦略を選択します。
3. 選択されたモード用に必要なフック/プロフィール を有効にします。
4. ループ計画を作成し、`.claude/plans/` 下のランブックを書きます。
5. ループを開始および監視するコマンドを出力します。

## 必須の安全チェック

- 最初のループイテレーション前にテストが通ることを検証します。
- `ECC_HOOK_PROFILE` がグローバルに無効化されていないことを確認します。
- ループに明示的な停止条件があることを確認します。

## 引数

$ARGUMENTS:
- `<pattern>`（`sequential|continuous-pr|rfc-dag|infinite` オプション）
- `--mode safe|fast`（オプション）
