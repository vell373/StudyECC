# PHP パターン

> このファイルは [common/patterns.md](../common/patterns.md) を PHP 固有のコンテンツで拡張します。

## 薄いコントローラー、明示的なサービス

- コントローラーを転送に焦点: 認証、検証、シリアライゼーション、ステータスコード。
- ビジネスルールをアプリケーション/ドメインサービスに移動して、HTTP ブートストラップなしでテスト可能にする。

## DTO と値オブジェクト

- shape 重いアソシアティブアレイを、リクエスト、コマンド、外部 API ペイロード用 DTO に置き換え。
- 値オブジェクトを金銭、識別子、日付範囲、その他制約付き概念に使用。

## 依存性注入

- フレームワークグローバルではなく、インターフェースまたは狭いサービスコントラクトに依存。
- サービスロケータールックアップなしでテスト可能なようにコンストラクタ経由でコラボレーターを渡す。

## 境界

- モデルレイヤーが永続性以上を行っている場合、ORM モデルをドメイン決定から分離。
- 小さなアダプタで第三者 SDK をラップして、残りのコードベースがあなたのコントラクトに依存し、彼らのコントラクトではなく。
